# BackyardHacks

**How to clone from branch:**
    ***git clone https://github.com/slee288/ru-hacks***  
**How to pull from current branch**
    ***git pull origin dev***  

**How to run the app**
    1. Move to your project folder, and run
    ***pip3 install -r requirements.txt***  
    2. Run the script
    ***python3 main.py***
